# Data Dictionary: `merged_data.csv`

This file describes the columns present in the final, processed dataset generated by this pipeline.

| Column Name | Data Type | Description | Source |
| :--- | :--- | :--- | :--- |
| **State_Name** | `String` | The full name of the State in question (ex. "California"). | Both (Merge Key) |
| **Year** | `Integer` | The observation year (2018, 2019, 2021, or 2022). | Both (Merge Key) |
| **Unemployment_Rate** | `Float` | The annual average unemployment rate for the state as a percentage. | FRED API |
| **FIPS_Code** | `String` | The 2-digit Federal Information Processing Standard (FIPS) code for the state, padded with leading zeros (ex. "06" for California). | U.S. Census API |
| **Median_Income** | `Float` | The estimated median household income in the state for the given year in nominal U.S. Dollars. | U.S. Census API |

**Notes on Data:**
* **Missing Years:** The year 2020 was explicitly excluded from this analysis due to extreme macroeconomic anomalies caused by the COVID-19 pandemic.
* **Unemployment Aggregation:** The raw FRED data is provided monthly. This pipeline aggregates those monthly rates into an annual mean to match the temporal granularity of the Census data.
